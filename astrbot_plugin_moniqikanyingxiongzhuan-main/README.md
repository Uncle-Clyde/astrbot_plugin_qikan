# astrbot_plugin_qiukan

骑砍英雄传（AstrBot 插件），提供聊天指令 + Web 双端玩法，包含战争、训练、突破、签到、挂机、历练、装备、家族、世界频道、排行榜、管理员面板等功能。

## 功能概览

| 模块 | 功能 |
| --- | --- |
| 战争成长 | 训练、突破（含爵位与道韵）、签到、挂机训练、装备/心法/功法、第纳尔上限 |
| 副本历练 | Web 端五层副本探索，天灾/魔物/敌人/在线遭遇战，回合制战斗，真实死亡与携宝重生 |
| 药剂系统 | 200 种药剂（5 品阶 × 3 品级），永久属性型 + 限时 buff 型，副作用机制 |
| 家族系统 | 创建/加入/退出/解散家族，职位管理（家主/副家主/长老/成员），家族仓库（上交/兑换/贡献点/角色折扣/规则配置） |
| 交易系统 | 集市上架/购买/下架，军需商店 |
| 社交系统 | 世界频道聊天（家族徽标）、排行榜（战争/死亡/在线） |
| 技能系统 | 骑砍风格技能树（被动技能 + 战斗技能），支持滚轮缩放和拖拽平移 |
| 管理后台 | 玩家资产编辑，武器/心法/功法/场景/公告/集市/IP 风控 CRUD |
| 渲染 | Pillow 图片消息（帮助/面板/背包/排行） |

## 主要配置

配置文件：`_conf_schema.json`

| 配置项 | 默认值 | 说明 |
| --- | --- | --- |
| `enable_web` | `false` | 是否启用 Web 服务 |
| `web_host` | `0.0.0.0` | 监听地址 |
| `web_port` | `8099` | 监听端口 |
| `web_access_password` | `""` | 访问密码，留空不校验 |
| `web_admin_account` | `admin` | 管理员账号 |
| `web_admin_password` | `""` | 管理员密码 |
| cultivate_cooldown | `60` | 训练冷却（秒） |
| afk_cultivate_max_minutes | `60` | 挂机上限（分钟） |
| adventure_cooldown | `1800` | 征战冷却（秒） |

## Web 使用

1. 开启 `enable_web` 后重启插件
2. 访问 `http://<host>:<port>`，注册/登录角色
3. 聊天端 `/骑砍 登录 <密钥>` 绑定 QQ 角色
4. Web 端支持副本、遭遇战、世界频道、家族仓库、集市、背包管理
5. 技能系统访问 `http://<host>:<port>/skills`
6. 管理员使用独立账号密码登录管理后台

## 骑砍技能系统

技能系统位于 `/skills` 页面，支持：

- **被动技能**：强击、铁骨、盾击、跑动、骑术、武器大师、教练、掠夺、手术、急救、统治术、工程学、说服力、战术、教练等 16 个技能树
- **战斗技能**：冲锋、盾击、战吼、狂暴、虚晃等
- **交互功能**：滚轮缩放、拖拽平移、技能筛选、搜索
