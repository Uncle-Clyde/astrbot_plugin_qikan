# astrbot_plugin_qikan

骑砍英雄传（AstrBot 插件），提供聊天指令 + Web 双端玩法，包含战斗、训练、等级、装备、家族、世界频道、排行榜、管理员面板等功能。

| Web UI 版本 | `0.1.1` |
| 技能系统 | `骑砍风格技能树` |
| 插件标识 | `astrbot_plugin_qikan` |
| 数据存储 | `SQLite`（`data/plugin_data/astrbot_plugin_qikan/xiuxian.db`） |

## 功能概览

| 模块 | 功能 |
| --- | --- |
| 角色成长 | 训练、升级、爵位晋升、签到、挂机训练、装备系统 |
| 战斗系统 | 劫匪战斗、副本历练、PVP对战 |
| 药剂系统 | 多种药剂，永久属性加成 + 限时 buff |
| 家族系统 | 创建/加入/退出/解散家族，职位管理，家族仓库 |
| 交易系统 | 集市上架/购买/下架，军需商店 |
| 社交系统 | 世界频道聊天、排行榜 |
| 技能系统 | 骑砍风格技能树（被动技能 + 战斗技能） |
| 管理后台 | 玩家资产编辑，装备/场景/公告/集市 CRUD |

## 主要配置

配置文件：`_conf_schema.json`

| 配置项 | 默认值 | 说明 |
| --- | --- | --- |
| `enable_web` | `false` | 是否启用 Web 服务 |
| `web_host` | `0.0.0.0` | 监听地址 |
| `web_port` | `8099` | 监听端口 |
| `web_access_password` | `""` | 访问密码，留空不校验 |
| `web_admin_account` | `admin` | 管理员账号 |
| `web_admin_password` | `""` | 管理员密码 |
| cultivate_cooldown | `60` | 训练冷却（秒） |
| afk_cultivate_max_minutes | `60` | 挂机上限（分钟） |
| adventure_cooldown | `1800` | 征战冷却（秒） |

| 指令 | 说明 |
| --- | --- |
| `帮助` | 显示所有可用指令 |

| `排行` / `死亡排行` / `在线` | 排行榜 |

## Web 使用

1. 开启 `enable_web` 后重启插件
2. 访问 `http://<host>:<port>`，注册/登录角色
3. 聊天端 `/骑砍 登录 <密钥>` 绑定 QQ 角色
4. Web 端支持副本、世界频道、家族仓库、集市、背包管理
5. 技能系统访问 `http://<host>:<port>/skills`
6. 管理员使用独立账号密码登录管理后台

## 骑砍技能系统

技能系统位于 `/skills` 页面，支持：

- **被动技能**：强击、铁骨、盾击、跑动、骑术、武器大师、教练、掠夺、手术、急救、统治术、工程学、说服力、战术等
- **战斗技能**：冲锋、盾击、战吼、狂暴、虚晃等
- **交互功能**：滚轮缩放、拖拽平移、技能筛选、搜索
